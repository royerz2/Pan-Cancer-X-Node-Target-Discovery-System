# Publication Checklist

## Pre-submission

- [x] README with installation and usage
- [x] requirements.txt with pinned versions
- [x] Benchmark against gold standard (61% recall)
- [x] Baseline comparison (random: 21%, top-genes: 39%)
- [x] Publication figures (benchmark, target frequency, triple patterns, synergy-resistance)
- [x] Manuscript outline (MANUSCRIPT_OUTLINE.md)
- [x] Data availability statement (DATA_AVAILABILITY.md)
- [x] Reproducibility script (run_full_pipeline.sh)

## Recommended before submission

- [ ] GitHub repository (public)
- [ ] Zenodo DOI for code
- [ ] In vitro validation of 1-2 top combinations
- [ ] Supplementary Table S1: Full triple combinations
- [ ] Supplementary Table S2: Gold standard benchmark details
- [ ] Supplementary Table S3: Novel combinations for lab testing
